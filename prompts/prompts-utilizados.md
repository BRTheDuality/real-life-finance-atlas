# Prompts utilizados no estudo

Este projeto utilizou prompts no NotebookLM para analisar documentos sobre educação financeira.

## Prompt 1
Explique os principais conceitos financeiros apresentados nos documentos de forma simples.

## Prompt 2
Quais são os conceitos mais importantes presentes nos documentos relacionados a planejamento financeiro pessoal?

## Prompt 3
Faça um resumo estruturado dos principais temas financeiros abordados nas fontes.

## Prompt 4
Quais são os conceitos fundamentais que uma pessoa precisa entender para tomar boas decisões financeiras?

## Prompt 5
Quais princípios financeiros são mais importantes para manter estabilidade econômica?

## Prompt 6
Crie um exemplo de planejamento financeiro para uma pessoa que ganha salário mínimo.

## Prompt 7
Quais riscos financeiros podem comprometer a estabilidade de uma família?

## Prompt 8
Quais fatores financeiros devem ser considerados antes de ter filhos?
